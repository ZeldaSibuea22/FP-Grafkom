# About
Dalam project ini,kami membuat permainan balap mobil, yang terdapat rintangannya. Pemain dapat mengumpulkan skor sebanyak banyaknya untuk menang.

# Controls

- W/UpArrow maju
- S/DownArrow mundur
- A/LefArrow kiri
- D/RightArrow kanan
- ESC/SpaceBar to pause/main game
- M to mute/unmute suara
